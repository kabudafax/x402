### 部署步骤

1. **设置环境变量**
   ```bash
   cd contracts
   cp .env.example .env
   # 填入 PRIVATE_KEY 和合约地址
   ```

2. **部署合约**
   ```bash
   forge script scripts/Deploy.s.sol:DeployScript \
     --rpc-url monad_testnet \
     --broadcast \
     --verify
   ```

3. **配置后端和前端**
   - 更新合约地址
   - 配置数据库
   - 启动服务
